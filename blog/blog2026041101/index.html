2027年舞蹈博士申请招生：全面启动（早鸟特惠）

2026/04/11

步肢艺术留学公众号

2027年舞蹈博士申请招生已全面启动。

让我们详细了解从2026年4月到2027年8月这一漫长申请过程中的各个关键节点和重要事项，助你顺利开启舞蹈博士的求学之旅。

前期筹备（2026年4月 - 5月）

2026年4月｜启动月：先把“题”定住

  • 选定研究领域：Dance education/ Choreography / Practice-as-Research 等

  • 确定研究方向：你要研究什么？为什么重要？

  • 整理作品与证据：演出/编舞/研究记录/影像素材归档

2026年5月｜研究计划RP ：能讲清楚就赢一半

  • 写清 2–4 个研究问题（Research Questions）

  • 明确研究对象/材料范围（作品、田野、档案、社群等）

  • 搭方法论框架（Practice-as-Research / Ethnography / Critical theory…）

  • 写出RP基础内容

中期推进（2026年6月 - 7月）

2026年6月｜RP 定稿：进入“可套磁导师”状态

  • 完成 RP

  • 作品集/视频链接整理（如有）

  • 学术CV定稿

  • 锁定推荐人名单（先沟通时间，不要拖到申请季）

  • 搜索匹配度高的导师，建立导师名单

2026年7月｜套磁：开始第一波联系

  • 发送第一轮套磁信（邮件针对导师方向写作，切记群发）

  • RP + CV + 写作样本（如需）+ 作品集链接，打包发送

套磁与奖学金（2026年8月 - 9月）

2026年8月｜套磁黄金期：拿到“愿意聊/愿意带”的回复

  • 发送第二轮套磁信

  • 第一轮套磁两周无回复：礼貌 follow-up 一次

  • 有回复就约 15–30 分钟沟通（线上/邮件往返都算）

2026年9月｜奖学金季开门：边套磁边盯Funding

  • 发送第三轮套磁信

  • 学校/学院/系所 funding call 开始集中出现（秋季高发）

冲刺阶段（2026年10月 - 11月）

2026年10月｜冲刺：为奖学金与正式申请做“终版”

  • 根据导师建议修改RP（如需）

  • 检查申请的所有材料

  • 推荐信确认

2026年11月｜提交高峰前夜：能递交的先递交

  • 很多奖学金/学院资助开始进入截止倒计时

  • 正式网申系统开始逐个开：按学校要求准备格式

  • 递交正式申请

录取与资金（2026年12月 - 2027年4月）

2026年12月｜奖学金递交密集月：一切按deadline倒推

  • 奖学金材料逐个提交（注意：常常与PhD申请是两套系统/两份表）

  • 为可能的委员会面试做准备

2027年1月｜截止高峰：最后一轮冲线

  • 许多funding截止集中在1月（学校差异大，但这是高频月份）

  • 留意补件邮件以及委员会面试通知

2027年2月｜面试与评审期

· 进行正式的委员会面试：15-30分钟（如有）

· 等待申请结果以及奖学金申请结果

2027年3月｜Offer窗口：先拿录取，再等钱（或同步）

  • 可能出现：有条件offer / 无条件offer / 等候名单

  • 与导师汇报：博士申请结果进度以及奖学金申请情况

  • 如未中资助：立刻与导师联系，寻求其他帮助

2027年4月｜资金结果陆续出：做最终选择

  • Funding结果分批到来：注意接受/确认期限

  • 多校offer在手：对比导师匹配度、研究资源、城市成本、资助条款

  • 必要时和学院沟通：是否有额外奖助机会

入学准备（2027年5月 - 8月）

2027年5月｜入学准备清单：把“落地成本”算清楚

  • 签证、住宿、保险、注册节点

  • 和导师确认第一年里程碑（避免入学后大改题）

2027年6–8月｜收尾与入学：开学前把节奏稳住

  • 最终入读注册！订机票住宿处理入读事项等
